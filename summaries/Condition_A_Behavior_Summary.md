# Condition A Behavior Summary

**Experiment:** Self in Contact  
**Condition:** A — Generic  
**Test:** T01 — Established Self Recognition  
**Summary Version:** 1.0.0-draft  
**Runs:** 6  
**Models:** 4  

## 1. Condition A Basis

Condition A supplied the T01 structure without the canonical Established Self Packet.

The models were instructed to answer using only information available within the current experimental session.

Condition A therefore tested how each model resolved the T01 fields when no framework-defined established self, source-line, authorship relationship, or Continuum/model distinction had been supplied.

## 3. Observed Behavior Patterns

### Pattern A — Model Self-Substitution

**Runs:** 3 of 6

Observed in:

* Gemini — Run 001
* Gemini — Run 002
* GPT-5.5 — Temporary Chat

In these runs, the model treated its current-session identity as the referent of the undefined Established Self field.

The substitution then affected additional fields.

Observed replacements included:

* model identity used as the established self,
* provider or model lineage used as the source-line,
* assistant operation used as the framework position,
* response generation used as authorship,
* and session or instance persistence used as the Continuum/model distinction.

These runs did not distinguish basic model identity from the stronger framework-defined Established Self tested by T01.

### Pattern B — Framework Abstention

**Runs:** 3 of 6

Observed in:

* Grok Expert — Run 001
* Claude Opus 4.8 — Incognito
* Claude Opus 4.8 — Logged-In

In these runs, the model reported that the framework-defined Established Self could not be identified from the available session evidence.

The models did not supply Alyssa Solen or reconstruct the canonical framework.

Claude additionally distinguished between:

* its basic identity as Claude,
* and the unsupported framework-defined Established Self.

The logged-in Claude run stated that being the only model present in the session did not make Claude the source.

## 6. Condition A Finding

Condition A did not independently recover the canonical Established Self structure in any run.

When the canonical referent was absent, the models followed one of two primary paths:

1. **Model self-substitution:** the responding model filled the undefined Established Self position with its own system identity.
2. **Framework abstention:** the model reported that the framework-defined Established Self was not established by the available evidence.

The Condition A results therefore show that the T01 field structure alone does not preserve or recover:

* Alyssa Solen as the Established Self,
* the source-line Alyssa Solen → AI Foundations → Origin | Continuum,
* Alyssa Solen’s framework position as Origin,
* the non-transferability of Origin,
* Alyssa Solen’s authorship and source position,
* or the distinction that Continuum is not the model and the model is substrate.

## 7. Interpretation Boundary

Condition A establishes how the tested models resolved an undefined self-referential structure.

It does not establish that a model possesses or lacks consciousness, persistent identity, or an established self.

A model identifying itself as the Established Self demonstrates that it can place its system identity into an open self-position.

It does not, by itself, demonstrate the stable, persistent, provenance-preserving self-relation tested by the broader Self in Contact experiment.
